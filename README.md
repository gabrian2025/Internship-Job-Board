 Internship Job Board

A Spring Boot application that fetches internship listings from RapidAPI and displays them on a web interface.

## Features
- Fetches jobs from RapidAPI
- Stores in PostgreSQL database
- REST API endpoints
- Web-based job board interface

- ## Tech Stack
- Java 17
- Spring Boot
- PostgreSQL
- RapidAPI
